## ✨ 项目简介

**途伴 TourMate** 是一款面向 **多人旅行、家庭出游、朋友结伴** 的轻量级微信小程序。

它解决了多人出行时的核心痛点：**位置不同步、行程不统一、照片难归集、沟通成本高**。
通过 **6位房间号** 快速组队，让同行的伙伴能够在同一个空间内，同步定位、共享路线、打卡美景、协同行程，让旅行更简单、更有趣。

---

## 🌟 核心功能

### 1. 极速房间组队
- 🔢 **6位数字房间号**：一键创建/加入房间，私密高效，无需互加好友。
- 👥 **成员管理**：在线状态实时显示，支持成员管理与权限分配。

### 2. 隐私安全的位置共享
- 📍 **实时定位**：地图上聚合查看全队成员位置。
- 🔒 **隐私可控**：**用户可随时开启/关闭位置共享**，保护个人隐私，不想被看见时一键关闭。

### 3. 协作式路线规划
- 🗺️ **统一路线**：创建并共享多日行程，景点顺序可视化。
- ✏️ **权限编辑**：支持**精细化权限控制**，仅授权用户可编辑路线，防止误操作。
- 🧭 **导航辅助**：查看路线距离与预估时长，规划更科学。

### 4. 可视化行程进度
- 📊 **进度条展示**：今日行程一目了然，直观显示待出发、进行中、已完成状态。
- ✅ **全员同步**：成员打卡后，全队实时更新进度。

### 5. 景点美图归集
- 📸 **一键上传**：拍摄或从相册选择当前景点照片。
- 🖼️ **节点绑定**：照片自动关联对应景点/路线节点，不乱传。
- 💾 **路线相册**：查看整条路线的美图合集，定格美好瞬间。

### 6. 路线分享与复用
- 🔗 **一键分享**：分享优质路线给好友。
- ♻️ **直接复用**：收藏并生成自己的同款行程，省时省力。
